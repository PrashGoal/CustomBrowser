# Custom Browser
It is similar browser like a chrome, it helps to do all browser activities.

## Features

- Helps you to browse online.
- Faster than chrome

The application helps to browse anything in the internet.

## Installation
> Install the PyQt5
```pip install PyQt5```

```pip install pyQtWebEngine```
